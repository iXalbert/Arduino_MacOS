# Proiect Arduino & IoT (Mac M3 Max)

Acesta este un repository dedicat dezvoltării de proiecte embedded folosind kit-ul Arduino Uno R3 și modulul ESP8266 (NodeMCU).

## 🛠️ Hardware utilizat
* **Laptop:** MacBook Pro M3 Max
* **Plăci de dezvoltare:** * Arduino Uno R3 (pentru bazele electronicii)
    * NodeMCU ESP8266 (pentru proiecte Wi-Fi/IoT)
* **Componente cheie:** Ecran LCD1602, Senzor Ultrasonic, Servo SG90, Motor pas cu pas, Senzor DHT11.

## 🚀 Proiecte Planificate
1.  **01_Blink:** Testarea conexiunii cu Mac-ul.
2.  **02_Safe_IR:** Seif electronic cu parolă prin telecomandă.
3.  **03_Weather_Station:** Stație meteo cu afișaj pe LCD.
4.  **04_IoT_Dashboard:** Controlul componentelor prin Wi-Fi folosind ESP8266.

## 💻 Setup Mac M3 Max
* **IDE:** Arduino IDE 2.x (Apple Silicon)
* **Port Serial:** `/dev/cu.usbserial-XXX` sau `/dev/cu.usbmodemXXX`
* **Limbaje:** C++ (Arduino) și MicroPython (opțional pentru ESP8266)
